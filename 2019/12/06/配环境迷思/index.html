Jupyter in Python 3.8

升级Python 3.8以后，重新配Jupyter遇到了坑，拖到今天终于解决了。

一点人生经验：

  • 无脑Anaconda>>自己pip折腾

  • 搜索报错信息的正确打开方式：

    • 在读懂报错的基础上，摘取关键字搜

    报错信息如下：

  1. 一开始无脑搜"Jupyter NotImplementedError" ，无果

  2. 阅读traceback，定位到self.asyncio_loop.add_reader(fd, self._handle_events, fd, IOLoop.READ)，搜索这个函数，找到解决方案

    是由于 python3.8 asyncio 在 windows 上默认使用 ProactorEventLoop 造成的，而不是之前的 SelectorEventLoop。jupyter 依赖 tornado，而 tornado 在 window 上需要使用 SelectorEventLoop，所以产生这个报错
    ————————————————

  3. 解决方案

    修改d:\python\lib\site-packages\tornado\platform\asyncio.py

也就是在 import asyncio 之后，手动让 tornado 选择 SelectorEventLoop，这样 jupyter 就可以正常使用了
